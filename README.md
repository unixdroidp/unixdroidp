## 👋 A Little About Me

Hello! I'm **Pushpendra**, a passionate **Cyber Security Educator** and **Ethical Hacker** from India.
I love breaking things (ethically 😄), teaching others how to defend their systems, and automating
repetitive work with Python. I'm the founder of **UnixDroid**, where I create educational content
and security awareness material for students and professionals.

- 🔭 Currently working on **UnixDroid**, **Alpha Computer Institute** and **Raj Computers Academy**
- 🛡️ Specialising in **Penetration Testing**, **Red Teaming** and **Web Application Security**
- 📚 I share study material and notes for **CEH v13** and offensive security
- 🌱 Currently leveling up in **Cloud Security (AWS)**, **Active Directory attacks** and **Bug Bounty**
- 🤝 Open to **collaboration**, **freelance projects**, **full-time roles** and **training opportunities**
- ⚡ Fun fact: *Security is a process, not a product — and learning never stops.*
